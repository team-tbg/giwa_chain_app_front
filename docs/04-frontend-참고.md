# 프론트엔드 참고

## 벤치마크: unifi.me — "이자 붙는 스테이블코인 지갑"

> https://www.unifi.me/ · **"A stablecoin wallet designed for earning"**
> ⚠️ **참고(벤치마크)일 뿐, 그대로 복제 아님.** 특히 unifi가 하는 "확정 % 이자·크립토 용어 노출"은 우리 규칙([08](./08-dev-context-primer.md))상 하지 않는다.

스테이블코인(USDT) 예치에 이자를 주고 송금·리워드를 얹은 **소비자용 핀테크 지갑**. "디파이를 이자 지갑으로 포장해 대중에게" 접근하는 결이 나드리와 동일.

### 실제 홈 구조 (2026-07 캡처 기준)

- **상단**: 로고(좌) · `로그인·회원가입`(우) · 플로팅 pill `최대 8% 이자 받아요!`
- **히어로**(그린 그라디언트 + 코인 일러스트): "매일 이자가 쌓이는 새로운 예치 경험 / 간편한 은행 계좌 송금 지원" + 검정 CTA `Unifi 알아보고 시작하기`
- **부스트 배너**: `⚡ KAIA 부스트로 최대 3% 추가 이자 받기 ›`
- **보너스 리워드 받기**: "매일 USDT 리워드 받기 · 출석체크부터 게임미션까지"
- **혜택 확인하기**: "역대급 이율 혜택", "특별 레퍼럴 랭킹 혜택 · 친구 초대하고 리워드 🏆"
- **이자 더 받기** 섹션
- **교육 카드 "제대로 알고 쓰기"**(2열 그리드): USDT/스테이블코인/비수탁 지갑/이자 운용/수익 내는 법/자산 옮기기
- **FAQ 아코디언**: 예치 시작법 · 지갑 입금법 · 복리 여부 · 기본/플러스 이율
- **하단 탭**: `홈` · `리워드` · `내 자산` · `마이`

### 우리가 빌릴 것 (○) / 다르게 갈 것 (×)

| unifi 방식 | 나드리 판단 |
|-----------|-------------|
| 홈=히어로+CTA, 이자 예치를 전면에 | ○ 구조 참고 |
| 출석체크·게임미션으로 매일 리워드 | ○ (걷기 외 트리거: 잠금해제·출석 등과 연결) |
| 레퍼럴 랭킹(친구 초대 리워드) | ○ 리텐션/바이럴로 후속 |
| 하단 탭 4개(홈/리워드/내 자산/마이) | ○ 참고 — 현재 3탭(홈/불리기/내 정보), "내 자산" 분리 검토 |
| **확정 % 노출**(8%/3%/기본·플러스 이율) | ✕ **금지** → "매일 조금씩 · 달라질 수 있어요"(원 단위 카운트업) |
| **크립토 용어 노출**(USDT·스테이블코인·비수탁지갑) | ✕ 더 숨김 — 우리 타깃은 용어 자체가 진입장벽 |
| 교육 카드로 크립토 학습 유도 | △ 선택형 아코디언으로 최소화(궁금한 사람만) |
| 은행 계좌 송금 | △ 후속(출금=통장 입금으로 이미 반영) |

> 핵심 차용점: ① **APY를 퍼센트가 아니라 "매일 붙는 금액"으로 시각화** ② 예치/출금을 **넣기·빼기 두 버튼**으로 단순화 ③ 체인/지갑/가스 완전 제거.
> 출처: [Unifi](https://www.unifi.me/), [Unifi Protocol 개요(Gate)](https://www.gate.com/learn/articles/what-is-unifi-protocol-dao/746)
