# GIWA 체인 소개

> 출처: https://docs.giwa.io/ (개발자 문서)

## 한 줄 정의

GIWA 체인은 **OP 스택(Optimism) 기반 이더리움 레이어 2(L2) 블록체인**이다.
Web3의 진입 장벽을 낮추고 접근성을 높이는 것을 목표로 한다.

## 핵심 특징

| 항목 | 내용 |
|------|------|
| 기반 기술 | Optimism 재단의 **OP Stack** |
| 유형 | 이더리움 L2 (Rollup) |
| 블록 타임 | 약 **1초** (빠른 블록 생성) |
| 실행 환경 | **EVM 호환** — 기존 Solidity 스마트 컨트랙트 그대로 사용 가능 |
| 특징 | 국내 개발자 대상 **한국어 문서** 제공, 간단한 온보딩 |

## 아키텍처 (L2 동작 방식)

- 별도 체인(L2)에서 트랜잭션을 처리한 뒤, **데이터와 결과만 이더리움(L1)에 기록**하여 속도·비용 개선
- OP Stack 기반이므로 Optimism/Base 등과 유사한 개발 경험 및 툴체인 활용 가능

## 개발자 관점 장점

1. **개발 편의성**: 기존 EVM/Solidity 도구(Foundry, Hardhat, viem/wagmi 등) 그대로 사용
2. **한국어 지원**: 국내 개발자 온보딩 용이
3. **간단한 배포**: 복잡한 서버 구축 없이 직접 배포 가능

## 정체성·비전

- **이름 철학**: 한국 전통 건축의 "기와"처럼, 개별 빌더가 모여 견고한 생태계를 형성
- **개방성**: 특정 주체에 귀속되지 않은 글로벌 인프라 지향
- **업비트 연계**: 신뢰받는 거래소와의 연결로 생태계 강화

## 프론트엔드 개발 시 참고

- OP Stack L2이므로 **표준 EVM 월렛/라이브러리**(wagmi, viem, ethers, RainbowKit 등) 그대로 적용 가능
- 체인 연동 시 GIWA의 **RPC 엔드포인트, chainId, 익스플로러 URL** 필요 → `docs.giwa.io`에서 네트워크 파라미터 확인
- L1↔L2 브릿지/출금 지연 등 롤업 특성을 UX에 반영(출금 대기 안내 등)
